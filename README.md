# 🛡️ Mini SOC System with Suricata IDS/IPS

## 📌 Overview

This project simulates a Security Operations Center (SOC) environment capable of monitoring, detecting, and responding to network attacks in real time.

## 🚀 Features

* 🔍 Network traffic monitoring using Suricata IDS/IPS
* ⚠️ Detection of attacks (Port Scan, ICMP flood, SSH brute force)
* 🧠 Custom rule creation
* 📊 Log aggregation using ELK Stack (Elasticsearch, Logstash, Kibana)
* 🐳 Docker-based deployment
* 🎯 Attack simulation using Kali Linux

## 🏗️ Architecture

* Attacker Machine: Kali Linux
* Target Network: Ubuntu Server
* IDS/IPS: Suricata
* Logging System: ELK Stack
* Environment: VirtualBox

## 🛠 Tools Used

* Suricata
* ELK Stack
* Wireshark
* Nmap
* Kali Linux
* Docker

## 🎯 Goal

To design and implement a functional SOC system capable of detecting and analyzing real-world cyber attacks.
